# 🐾 贡献指南 (Contributing Guide)

感谢你对 **Paws-Places-DB** 开放数据库的关注！我们希望通过社区的力量，共同维护一份最准确、最全面的深圳宠物友好地点清单。

## 如何贡献数据？

我们采用 GitHub Pull Request (PR) 的方式来接收新的地点提交或信息更正。

### 1. 提交新地点
1.  **Fork** 本仓库。
2.  打开 `data/places.json` 文件。
3.  在数组末尾添加一个新的 JSON 对象（注意：请不要包含 `_id` 字段，该字段由数据库自动分配）。
4.  参考下方的[数据格式规范](#数据格式规范)填写字段。
5.  提交更改并点击 **Create Pull Request**。

### 2. 修正现有信息
如果你发现某个商铺已经关门、地址搬迁或宠物政策有变，请直接修改 `data/places.json` 中对应的条目并提交 PR。

---

## 数据格式规范

提交数据时，请务必遵守以下字段格式：

```json
{
  "name": "商铺全称",
  "category": "分类（宠物店/商场/公园/咖啡厅/餐厅/创意园区）",
  "average_rating": 4.5,
  "address": "详细街道地址",
  "location": {
    "latitude": 22.543099,
    "longitude": 113.943102
  },
  "tags": {
    "friendly_type": ["宠物店"],
    "pet_restriction": ["仅限小型犬"],
    "access_rule": ["可下地"],
    "facilities": ["提供水碗"]
  }
}
```

### 注意事项：
*   **坐标准确性**：推荐使用腾讯地图坐标拾取器，确保经纬度准确到小数点后 6 位。
*   **标签选择**：请尽可能提供详细的 `tags`，这对其他宠主非常有帮助。
