# 🐾 Paws-Places-DB 零基础贡献指南

如果你想参与建设深圳最全的宠物友好地图，但不知道怎么操作，请看这份“手把手”教程！

---

## 选项 1：我完全不会写代码 (推荐！)
你只需要把信息告诉我们，我们来录入。
1.  在本页面顶部找到 **Issues** 菜单。
2.  点击 **New Issue**（新建议题）。
3.  像发帖一样，写下你推荐的店铺名字和地址。
4.  点提交。**大功告成！**

---

## 选项 2：我愿意尝试修改数据 (超赞！)
你不需要下载任何软件，在网页上就能完成。

### 第一步：准备数据
根据我们的格式规范（详见下方）准备好新店的信息。
> **💡 找经纬度小贴士**：打开 [腾讯地图坐标拾取器](https://lbs.qq.com/getPoint/)，搜索地点后即可在右上方看到一串数字。前面的是经度 (longitude)，后面的是纬度 (latitude)。

### 第二步：编辑文件
1.  在项目首页进入 `data/` 文件夹，点击 `places.json`。
2.  点击右上角的 **笔形图标 (Edit)**。
3.  滚动到文件的最底部。
4.  在倒数第二个 `}` 后面加一个英文逗号 `,`，然后粘贴新店的信息（注意：**千万不要包含 `_id` 字段**）。

### 第三步：保存并提交
1.  在页面下方写个简单的备注（例如：“添加了 某某咖啡店”）。
2.  点击绿色按钮 **Propose changes**。
3.  点击 **Create Pull Request**。

---

## 📋 数据格式与字段详细说明 (重点！)

为了保证你的推荐能在小程序上完美展示，请严格参考以下 JSON 格式。

```json
{
  "name": "商铺全称",
  "category": "咖啡厅",
  "address": "详细街道地址",
  "location": {
    "latitude": 22.543099,
    "longitude": 113.943102
  },
  "average_rating": 4.5,
  "business_hours": "周一至周日 10:00-22:00",
  "phone": "+86 13800138000",
  "description": "一段关于这家店对宠物有多友好的简短介绍...",
  "tags": {
    "friendly_type": ["咖啡厅"],
    "pet_restriction": ["犬猫友好", "仅限小型犬"],
    "access_rule": ["室内需牵绳", "可下地"],
    "facilities": ["提供水碗", "拾便袋"]
  },
  "avatar_image": "",
  "display_images": []
}
```

### 🔴 必须填写的核心字段 (缺失会导致小程序无法显示或报错)
根据前端代码的运行逻辑，以下四个字段是**绝对必须**的：

*   **`name` (必填 ⭐)**
    *   **中文含义**：店铺名称。
    *   **作用**：小程序列表和详情页的大标题，地图气泡上方显示的名字。
*   **`category` (必填 ⭐)**
    *   **中文含义**：店铺分类。
    *   **作用**：决定了地图上显示的专属图标（咖啡杯、骨头等）以及列表的分类过滤。
    *   **⚠️ 注意**：**必须且只能**从以下词中选择：`宠物店`、`商场`、`公园`、`咖啡厅`、`餐厅`、`创意园区`。
*   **`address` (必填 ⭐)**
    *   **中文含义**：详细地址。
    *   **作用**：用于在详情页展示给用户看，也是触发地图一键导航功能的重要信息。
*   **`location` (必填 ⭐)**
    *   **中文含义**：地图坐标。
    *   **作用**：包含 `latitude` (纬度) 和 `longitude` (经度)。这是最关键的字段，**如果没有坐标，地图上就不会出现定位点！** 强烈建议使用腾讯地图坐标拾取器获取精确的小数点后 6 位坐标。

### 🟡 强烈建议填写的体验字段 (能极大提升其他宠主体验)
*   **`average_rating`**：
    *   **中文含义**：综合评分。
    *   **输入内容**：填写 `1.0` 到 `5.0` 之间的数字。这会在详情页点亮对应数量的“小肉垫”图标。
*   **`description`**：
    *   **中文含义**：简短推荐语/简介。
    *   **输入内容**：说说为什么推荐这家店？保安态度好吗？有大草坪吗？
*   **`tags`**：
    *   **中文含义**：店铺标签。
    *   **输入内容**：分为四个小分类：
        *   `friendly_type`: 友好类型（如：宠物店、商场）
        *   `pet_restriction`: 宠物限制（如：犬猫友好、大型犬友好、仅限小型犬）
        *   `access_rule`: 出入规则（如：需要牵绳、需要推车、可下地、可松绳）
        *   `facilities`: 提供设施（如：宠物水碗、拾便袋、可租推车、停狗点）

### ⚪ 可选或预留字段
*   **`business_hours`**：
    *   **中文含义**：营业时间。
    *   **输入内容**：如 "09:00-21:30" 或 "全天"。
*   **`phone`**：
    *   **中文含义**：商铺联系电话。
    *   **输入内容**：方便用户直接在小程序里点击拨打。
*   **`avatar_image` / `display_images`**：
    *   **中文含义**：头像图片和展示图片预留字段。
    *   **输入内容**：因为开源代码不直接接受图片上传，你在提交时保持为空（`""` 或 `[]`）即可，后续管理员会统一上传真实的云端图片链接。
