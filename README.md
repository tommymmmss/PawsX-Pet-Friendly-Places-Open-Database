# 🐾 Paws-Places-DB: 深圳宠物友好空间开放数据库

本项目是一个致力于收集、整理和分享深圳地区“宠物友好（Pet-Friendly）”场所的开源数据库。

## 🎯 项目目标
通过社区共建的方式，维护一份最全面、最准确的宠物友好地点清单。数据以标准的 JSON 格式存储，方便开发者集成到各类小程序、网页或 APP 中。

## 📂 数据说明
核心数据存放在 [data/places.json](data/places.json) 中，包含以下信息：
*   **商铺名称**
*   **精确经纬度**（用于地图展示）
*   **宠物政策**（是否可下地、是否提供水碗、大小型犬限制等）
*   **营业时间及联系方式**

## 🤝 如何贡献数据？
我们非常欢迎你推荐身边的宠物友好去处！
请阅读我们的 [贡献指南 (CONTRIBUTING.md)](CONTRIBUTING.md) 了解如何提交你的第一条数据。
