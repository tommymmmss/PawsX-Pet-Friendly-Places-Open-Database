# 🐾 Paws-Places-DB: 深圳宠物友好空间开放数据库

本项目致力于收集深圳所有可以带毛孩子一起出入的宝藏地点。

**不论你是资深开发者，还是单纯想分享好店的宠主，都欢迎加入我们！**

---

## 👋 核心指南：如何推荐新地点？

为了让所有人都能参与，我们提供了三种难度的贡献方式：

### 🟢 简单模式：直接留言 (无需技术背景)
如果你不想处理代码，只需 1 分钟：
1.  点击页面上方的 [Issues (议题)](../../issues) 标签。
2.  点击绿色按钮 **New Issue**。
3.  在标题写上“推荐新地点：店名”，在正文里写下店名、地址和你的简单评价。
4.  点击 **Submit**。我们会替你把信息转录成数据库格式。

### 🟡 进阶模式：网页快速修改 (无需下载工具)
如果你想让自己的名字出现在贡献者名单里：
1.  打开 [data/places.json](data/places.json) 文件。
2.  点击右上角的 **笔状图标 (Edit this file)**。
3.  模仿最后一行的数据，把新店信息复制并粘贴在数组末尾。
4.  滚动到底部，点击 **Propose changes**。

### 🔴 专业模式：GitHub PR
适合开发者，请参考 [详细贡献指南 (CONTRIBUTING.md)](CONTRIBUTING.md)。

---

## 📂 数据结构说明
如果你选择自己修改文件，请确保包含以下关键信息：
*   **name**: 店铺名字
*   **address**: 详细地址
*   **category**: 分类（咖啡厅/餐厅/商场/公园/宠物店）
*   **location**: 经纬度（可以在腾讯地图搜索后右键获取）
